# Contributing

Teşekkürler! Bu proje küçük ve odaklıdır — katkılar memnuniyetle karşılanır.

- Hataları (issues) açık ve açıklayıcı biçimde bildirin.
- Yeni özellik talepleri için bir issue açın, kullanım senaryosunu ve örnek dosya(ları) ekleyin.
- Küçük değişiklikler için doğrudan pull request gönderin; büyük değişiklikler öncesi bir issue ile tartışma başlatın.
- Kod formatı: Python kodu için PEP8 uygundur. Linter veya format aracı eklemeyi düşünürseniz PR içinde belirtin.

Geliştirme ortamı

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
